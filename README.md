# Angular Chat Application
Angular + MongoDB + socket.io

## To use:
* Clone the repo: `git clone https://github.com/bayoishola20/angular-chatApp.git`

* `npm install`



## You should have installed (most recent versions):

* Node
* MongoDB localhost or use mlab
* Express
* socket.io
